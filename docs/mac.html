Limitations

  • ⚠ Password fields: macOS enables Secure Input for password fields, which bypasses keyboard listeners entirely тАФ Accentra (correctly) cannot see or modify what you type there.
  • ⚠ Key auto-repeat: While Accentra is running, holding a key with accent variants enters accent mode instead of repeating the character. Keys without variants repeat normally.
  • ⚠ Native picker popup: The macOS accent picker may appear while you hold a key. It's harmless тАФ Accentra's cycling works regardless тАФ but it is visible.
  • ⚠ Touch Bar / dictation / on-screen keyboard: Accentra intercepts physical keyboard input only.

Customizing accent maps

Accentra's accent sequences are fully configurable. To open the accent maps folder, click the ─Б menu bar icon and choose Edit accent maps.... Open accent-maps.json with any text editor тАФ each entry maps a base letter to a string of variants:

{
  "e": "├й├и├к├л─Ыe",
  "c": "─Н─З├зc",
  "s": "┼б┼Ы┼Яs"
}

Each character in the string becomes one step in the cycle. The last character is typically the plain letter itself, which lets you cycle back to the start. You can add new base keys, remove ones you never use, or reorder variants to put your most-used accents first. Changes take effect immediately тАФ no restart needed.
