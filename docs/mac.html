How it works

Accentra sits in your menu bar and watches your keyboard. When you hold a key a little longer than usual, it enters accent mode — no popup, no overlay, purely keyboard-driven.

1

Hold the key

Hold e for ~0.5 s. Accentra silently replaces it with é.

2

Tap to cycle

Each tap of e advances through é → è → ê → ë → e.

3

Confirm automatically

Press any other key or wait 1 second — accent mode exits and typing continues normally.

e (hold) → é    e (tap) → è    e (tap) → ê
s (hold) → š    c (hold) → č    z (hold) → ž

Works in Safari, Messages, Discord, browsers, TextEdit — everywhere a physical keyboard types.

Because accent maps are fully customizable, you can go beyond language accents. Add Greek letters, math operators, or any Unicode symbol — and type them with the same hold-and-cycle gesture:

# with a custom math section enabled:
f (hold) → ∀    e (hold) → ∈    i (hold) → ∩
# result:  (∀x)(x ∈ A ∩ B)

Plays nicely with the macOS accent picker

macOS has its own press-and-hold accent picker. Accentra takes over only the keys in its accent map: hold one of those and Accentra's replacement and cycling handle it — the native picker doesn't pop up, so the two never collide. Every other key is left completely alone, so the macOS picker still works as usual for anything Accentra doesn't map. Accentra does this per-keystroke, the moment you hold a key; it never changes the system-wide press-and-hold setting, and pausing or quitting hands every key straight back to macOS.

Pausing Accentra (from the menu bar) instantly gives you the stock macOS behavior in every app — no relaunches, no settings changed. Quitting leaves your system exactly as macOS shipped it.

Installation

Use the Download button at the top of this page — it's right for any Mac from the last few years. Only on an older Intel Mac (2020 or earlier) use the “older Intel Mac” link instead. Not sure? Open the Apple menu → About This Mac: if it lists a “Chip”, use the main Download button; if it lists a “Processor”, use the Intel link. Then:

Open the DMG and drag to Applications

Double-click the downloaded .dmg. A window opens showing Accentra and an Applications shortcut — drag Accentra onto Applications. When it's done, eject the “Accentra” disk from your Desktop or Finder sidebar (drag it to the Trash or press ⌘E).

Open Accentra

Launch it from your Applications folder. The first time, macOS asks you to confirm opening an app downloaded from the internet — click Open. (Accentra is signed with an Apple Developer ID and notarized, so there's no “unidentified developer” warning.)

Grant Accessibility access

The ā icon appears in your menu bar right away — dimmed, with a ⚠ Enable Accessibility access… item in its menu (macOS requires this permission for any app that listens to the keyboard). Click it to jump straight to System Settings → Privacy & Security → Accessibility and switch Accentra on.

Done — no relaunch needed

The moment you flip the switch, Accentra activates itself: the icon brightens and a “Accentra is ready” confirmation appears. No need to quit and reopen. Accentra also enables Start at Login on first run, so it's ready in your menu bar every time you log in — you can turn that off any time from the menu.

Prefer Homebrew? Install and update from the command line:
brew tap ipavicevic/tap brew trust --cask ipavicevic/tap/accentra brew install --cask accentra
The brew trust line is a one-time step Homebrew requires before installing a cask from a third-party tap. Or build from source with the .NET SDK: git clone https://github.com/ipavicevic/Accentra, then dotnet run --project Accentra.Mac/Accentra.Mac.csproj.

Menu bar menu

Click the ā icon in the menu bar for options:

  • Start at Login — toggle whether Accentra launches automatically when you log in.
  • Pause accent input — temporarily hand the keyboard back to macOS without quitting; the native press-and-hold picker behaves as stock.
  • Sections — enable or disable individual accent groups directly from the submenu.
  • Edit accent maps… — opens the accent maps folder in Finder so you can edit accent-maps.json directly.
  • Open log file… — opens Accentra's local log, useful when reporting a problem.
  • About Accentra… — opens this web page. The menu item shows the installed version.
  • Quit — stop Accentra until the next login (or until you run it again).

Limitations

  • ⚠ Password fields: macOS enables Secure Input for password fields, which bypasses keyboard listeners entirely — Accentra (correctly) cannot see or modify what you type there.
  • ⚠ Key auto-repeat: While Accentra is running, holding a key in its accent map enters accent mode instead of repeating the character. Keys Accentra doesn't map repeat (or show the macOS picker) as normal.
  • ⚠ Touch Bar / dictation / on-screen keyboard: Accentra intercepts physical keyboard input only.

Customizing accent maps

Accentra's accent sequences are fully configurable. To open the accent maps folder, click the ā menu bar icon and choose Edit accent maps.... Open accent-maps.json with any text editor — each entry maps a base letter to a string of variants:

{
  "e": "éèêëěe",
  "c": "čćçc",
  "s": "šśşs"
}

Each character in the string becomes one step in the cycle. The last character is typically the plain letter itself, which lets you cycle back to the start. You can add new base keys, remove ones you never use, or reorder variants to put your most-used accents first. Changes take effect immediately — no restart needed.

Privacy Policy

Accentra does not collect, store, or transmit any personal data.

  • ✓ No keylogging: Keystrokes are processed locally in real time solely to detect long-press timing and inject accented characters. They are never recorded or stored.
  • ✓ Never sees your passwords: macOS turns on Secure Input for password fields, which hides your typing from every keyboard utility — Accentra included. Accentra never receives, sees, or modifies what you type into a standard password field.
  • ✓ No network access: Accentra makes no network connections of any kind.
  • ✓ No telemetry: No usage data, crash reports, or analytics are collected.
  • ✓ No third-party services: Accentra has no dependencies on external services or APIs.
  • ✓ Local data only: The only files written to disk are accent-maps.json (your accent configuration) and a local diagnostic log — both stay on your Mac.

Known issues
