How it works

Accentra sits in your system tray and watches your keyboard. When you hold a key a little longer than usual, it enters accent mode — no popup, no overlay, purely keyboard-driven.

1

Hold the key

Hold e for ~0.5 s. Accentra silently replaces it with é.

2

Tap to cycle

Each tap of e advances through é → è → ê → ë → e.

3

Confirm automatically

Press any other key or wait 1 second — accent mode exits and typing continues normally.

e (hold) → é    e (tap) → è    e (tap) → ê
s (hold) → š    c (hold) → č    z (hold) → ž

Works in Discord, Messenger, browsers, Notepad — everywhere. No administrator rights required.

Because accent maps are fully customizable, you can go beyond language accents. Add Greek letters, math operators, or any Unicode symbol — and type them with the same hold-and-cycle gesture:

# with a custom math section enabled:
f (hold) → ∀    e (hold) → ∈    i (hold) → ∩
# result:  (∀x)(x ∈ A ∩ B)

Installation

Download Accentra.exe

Click the download button above. Save the file anywhere — your Downloads folder is fine.

Run it once

Double-click Accentra.exe. It copies itself to %LOCALAPPDATA%\Accentra\, registers with Windows so it starts automatically on login, and appears in Settings → Apps → Installed apps. After the first run, the file you downloaded is no longer needed and can be deleted.

Windows may show a "Windows protected your PC" warning because Accentra is not yet code-signed. This is expected. Click More info, then Run anyway to proceed.

Done

Accentra is running and will start automatically every time you log in. The ā icon appears in your system tray — Windows places new tray apps in the hidden overflow area (the ^ arrow near the clock) by default.

Don't see the icon? Click the ^ arrow in the taskbar corner to expand hidden tray icons. To keep it always visible, go to Settings → Taskbar → Other system tray icons and turn on Accentra.
Uninstalling: Go to Settings → Apps → Installed apps, search for Accentra, and click Uninstall. It removes itself completely — no leftovers.

Tray menu

Right-click the ā icon in the system tray for options:

  • Start with Windows — toggle whether Accentra launches automatically on login (enabled by default after install).
  • Pause accent input — temporarily disable accent mode without quitting; all keystrokes pass through normally until you unpause.
  • Sections… — open the Sections dialog to enable or disable individual accent groups (Latin, Slavic, Nordic, …).
  • Edit accent maps… — opens the accent maps folder in Explorer so you can edit accent-maps.json directly.
  • About Accentra… — opens this web page.
  • Quit — stop Accentra until the next login (or until you run it again).

Running as Admin

By default Accentra runs in non-privileged mode, so by Windows design it can't intercept keystrokes within apps running elevated.
To work around this limitation, open a command window as admin and run Accentra. Because Accentra is on your PATH, it will start elevated and become available in elevated apps.

Limitations

  • ⚠ Touch / soft keyboard: Accentra intercepts physical keyboard input only. The Windows on-screen keyboard and touch input use a different path that bypasses the hook entirely.
  • ⚠ Elevated apps: Accentra does not intercept keystrokes in apps running as Administrator. See Running as Admin for a workaround.
  • ⚠ Key auto-repeat: While Accentra is running, holding a key enters accent mode instead of producing repeated characters. Auto-repeat is not available for keys that have accent variants defined.

Customizing accent maps

Accentra's accent sequences are fully configurable. To open the accent maps folder, right-click the ā tray icon and choose Edit accent maps.... Open accent-maps.json with Notepad or any text editor — each entry maps a base letter to a string of variants:

{
  "e": "éèêëěe",
  "c": "čćçc",
  "s": "šśşs"
}

Each character in the string becomes one step in the cycle. The last character is typically the plain letter itself, which lets you cycle back to the start. You can add new base keys, remove ones you never use, or reorder variants to put your most-used accents first. Changes take effect immediately — no restart needed.

Privacy Policy

Accentra does not collect, store, or transmit any personal data.

  • ✓ No keylogging: Keystrokes are processed locally in real time solely to detect long-press timing and inject accented characters. They are never recorded or stored.
  • ✓ No network access: Accentra makes no network connections of any kind.
  • ✓ No telemetry: No usage data, crash reports, or analytics are collected.
  • ✓ No third-party services: Accentra has no dependencies on external services or APIs.
  • ✓ Local data only: The only file written to disk is accent-maps.json, which contains your accent configuration and nothing else.
